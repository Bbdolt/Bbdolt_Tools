TomcatScanPro 使用手册

项目地址: https://github.com/lizhianyuguangming/TomcatScanPro

准备工作

  • 准备包含URL、用户名和密码的文本文件
  • 分别命名为：
    • urls.txt（需自定义）
    • user.txt（已存在可自定义）
    • passwd.txt（已存在可自定义）
  • urls.txt 保存格式：
    https://127.0.0.1/ 或者 https://127.0.0.1/manager/html
    脚本会自行判断检测

配置文件设置

在 config.yaml 中配置文件路径和其他设置。

# config.yaml 示例
uris_file: "urls.txt"
users_file: "user.txt"
passwords_file: "passwd.txt"
threads: 10
timeout: 10

执行扫描

运行脚本：

python TomcatScanPro.py

脚本执行后，成功利用的漏洞信息将会记录在 success.txt 文件中。
